You can help me by the coordinate of spawn location of contracts (Drones, Power Up, Drill, White crystals) on both map.
You can do that by :

  1. Enable debug mode

  2. Add debug icon on the map

  3. Move it to a place (by drag and drop)

  4. Save debug icon coordinate

  5. Get coordonate in console (CTRL+SHIFT+I)

  6. Right click, and save as

  7. Save and send me the log file
